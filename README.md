# LesbianFlag2D

A 2D visualization of the Lesbian Pride Flag using C programming.
### Note
⚠️ To use this program, ensure that Raylib is installed on your system.

#### Debian

```sh

cd LesbianFlag2D

# Compile the program
make

# Run the executable
./LesbianFlag2D
